# Soccer Physics Engine

A physics-based soccer analytics platform that extracts tactical intelligence, biomechanical load monitoring, and player movement insights from tracking data.

**Status:** Under active development.

The goal is to turn tracking data into practical tactical and performance insights through a shared physics-first pipeline.

## Tech Stack
Python · NumPy · SciPy · PyTorch · FastAPI · Docker · AWS · React · D3.js

## Data
- Metrica Sports open tracking data (25fps, 22 players + ball)
- StatsBomb open event data
